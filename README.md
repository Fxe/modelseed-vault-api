# ModelSEED Annotation API

A Python library for interacting with ModelSEED annotation services.

## Installation

```bash
pip install modelseed_annotation_api
```

## Usage

```python
from modelseed_annotation_api import ModelSEEDAnnotation

# Initialize the client
client = ModelSEEDAnnotation()

# Use the client to interact with ModelSEED services
# Example methods will be added here
```

## Development

To set up the development environment:

```bash
git clone https://github.com/yourusername/modelseed_annotation_api.git
cd modelseed_annotation_api
pip install -e .
```
